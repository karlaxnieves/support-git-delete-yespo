Widgets gamification uses gaming elements in *Subscription* and *Request* *forms* to increase user engagement and motivation.Â 

Yespo offers three gaming mechanics available to subscribers after providing their email addresses (or other contact information):

1. **Spin the wheel**

The user spins a wheel with sectors, each corresponding to a prize or bonus, such as a discount, a gift, or bonus points.

<Image align="center" width="80% " src="https://files.readme.io/0cf0a433fe54333e7c1079a34bf4674aef5ff6a57339c53c398a02b165c81b56-widgets-gamification-001.gif" />

2. **Gift box**

The user selects one or several gifts hidden under images. The gift opens after clicking, and the user receives a prize, such as a discount, bonus points, or a special offer.

<Image align="center" width="80% " src="https://files.readme.io/c015e824e3333e217f79b3c40105daec4286da98d8a79412f3fdd66a161f7a48-widgets-gamification-002.gif" />

3. **Scratch card**

The user scratches the protective field, like lottery cards, to reveal their prize.

<Image align="center" width="80% " src="https://files.readme.io/59f2c61ea8aa6fb987cabe8de046a2a219d7675d95978b87e8d9a1e8ef2ae507-widgets-gamification-003.gif" />

### General Tab

Set the rewards parameters for all gamified elements on this tab:

1. Click the *Edit rewards* button.

<Image align="center" width="80% " src="https://files.readme.io/296d2083eede7480d293b3fe8660e3ea8fda406a2cac2f627e8440b8380fe072-widgets-gamification-013.webp" />

2. Set the parameter values:

* **Reward type** â€“ percentage discount.
* **Chance to win** â€“ the probability of receiving a reward.
* **Details** â€“ information about the reward in the interactive component.
* **Promo code** that the user receives at the specified email address or phone number.

<Image align="center" width="80% " src="https://files.readme.io/beeecb0900e5f9896ffa7e836fc95cc8f48a775d0a44b44297c5cc1af290169b-widgets-gamification-014.webp" />

3. If needed, add the number of rewards in the widget â€” click the *Add reward* button and select the reward type from the dropdown menu:

* **Percentage discount** â€“ in the form of a specific percentage, such as 10%, 20%, or 50%.
* **Custom** â€“ a unique gift, such as a product, certificate, bonus points, or another reward.

<Image align="center" width="80% " src="https://files.readme.io/1d072359fb9a2a5ff064e1a06ba63f6eb2d5e0dfdab90318f4eba3d3551357db-widgets-gamification-015.webp" />

For the percentage discount, the description is displayed as numbers, while for a custom reward, you can add any text that will appear in the widget.

#### Equalize

Use equalizing to automatically set an equal probability of winning for each reward in the element.

<Image align="center" width="80% " src="https://files.readme.io/fe6faa3baebe76b014d291b355af47418ef6a6dcc2e3f590e779213664d63ac8-widgets-gamification-017.webp" />
