## Blocks Configuration

Drag the blocks to the pre-created product card and click on those you want to edit â€” a settings panel will open on the left.

### Price Block

You can change the following settings in the block:

#### Show old price

This is the default option. Deactivate the slider to display only the current price.

<Image align="center" width="80% " src="https://files.readme.io/a6df132154c2cd1668903fb6da1bece70357a36cf26f771f5a083fc3598cf6c6-show-old-price-en.png" />

#### Distance between prices

Set the distance between the price and old price in pixels. The maximum value is 20 px.

<Image align="center" width="80% " src="https://files.readme.io/2daec15911a279f3b32d863d1843a392adf75f3aa626c75b63064244ec550e03-distance-between-prices-en.png" />

When selecting the **Maximum** option, the prices within the block will be spaced as far apart as possible.

<Image align="center" width="80% " src="https://files.readme.io/1f2ca508658a7e4227d1d5dfef504ab12d2b6384c1aa43d5d065f3d643b9a73d-prices-max-distance-en.png" />

#### Alignment

Set the alignment of the prices: left, center or right.

<Image align="center" width="80% " src="https://files.readme.io/9dc62705cf33145c679cf5bc4ed416643ead72605a5716ab3970a49ba7c80c46-proce-alignment-en.png" />

#### Price

You can set the following options in the **Price** and **Old price** sections:

1. Color, size, and font style.

<Image align="center" width="80% " src="https://files.readme.io/995e5ad8f984a2af375854c2c08413b220fe0ae01b2d395477b420066e6f500a-price-font-en.png" />

2. Positioning the old price in relation to the new price: left, right, below or above.

<Image align="center" width="80% " src="https://files.readme.io/4d3e03af84da8424c0467f54cc92c277fb152093285f7b95a06bb10462f62ab3-old-price-right-en.png" />

#### Currency

Specify the font size separately in pixels, or match it to the price size.

<Image align="center" width="80% " src="https://files.readme.io/d4df94624e45aaba55d564e0a555339b91a9d6cb47794bcf02231314cc0c1d61-price-currency-text-size-en.png" />

#### Paddings

The **Paddings** options allow adjustment of space around the desktop or mobile versions.

You can set all margin values at once or individually for each side of the block. The maximum value is 40 px.Â 

To set up the individual value for each side of the block, enable the **More** toggle and set the value.

<Image align="center" width="80% " src="https://files.readme.io/d062f7dcd61a4f0c7601c2e3ac2c13a97b67774bfe0811fcb5f74706cd6895a7-price-puddings-en.png" />

Click on the **smartphone** icon to set the paddings for the mobile version.

<Image align="center" width="80% " src="https://files.readme.io/647f2fcbd94b444a617bba17d39bfd653a5fd0f5f9a52548ccaca5378aca4341-price-puddings-mobile-en.png" />

### Discount Block

Changing the settings is similar to the **Price** block.

You can configure the following block settings:

* Alignment
* Font size and style
* Text color
* Background color
* Rounding corners
* Internal and external paddings around the block for mobile and desktop versions
